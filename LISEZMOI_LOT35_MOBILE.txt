1) PHOTO VÉHICULE DEPUIS LE TÉLÉPHONE
───────────────────────────────────────────────────────────────
Nouveau bouton « Photo » sur chaque carte véhicule (mobile).
  • Sélection depuis la galerie OU prise de photo (appareil photo) :
    input accept="image/*" sans capture → le système propose les
    deux (iPhone Safari + Android Chrome).
  • Aperçu avant enregistrement, avec « Reprendre / changer ».
  • Compression/redimensionnement automatique (max 1280 px, JPEG ~82%)
    via le mécanisme existant ASLDB.uploadImage.
  • Enregistrement → /api/upload (KV) puis mise à jour du véhicule
    (img/image/photo) → visible immédiatement côté back-office ET
    côté site client, et persistant après rechargement/reconnexion.
  • Ne casse pas la mise en page ; aperçu borné (max 46% hauteur écran).
  Note : le téléversement nécessite le site déployé sur Cloudflare ;
  en local (file://) un message clair s'affiche.
